◆ force_main_equation

core.swmm.options.dynamic_wave.DynamicWave.force_main_equation

Establishes whether the Hazen-Williams (H-W) or the Darcy-Weisbach (D-W) equation will be used to compute friction losses for pressurized flow in conduits that have been assigned a Circular Force Main cross-section shape.

The default is H-W.

◆ head_tolerance

core.swmm.options.dynamic_wave.DynamicWave.head_tolerance

Difference in computed head at each node between successive trials below which the flow solution for the current time step is assumed to have converged.

The default tolerance is 0.005 ft (0.0015 m).
